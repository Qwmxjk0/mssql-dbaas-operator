# Documentation Index

## Core documents
- `mssql-operator-rbac-contract.th.md`: RBAC responsibility split (DB team vs Core team)
- `mssql-core-migration-test-runbook.md`: step-by-step migration and verification runbook

## How to use this folder
1. Read RBAC contract first and align ownership with Core team.
2. Run migration test runbook in a fresh namespace.
3. Complete production gates checklist with evidence links.
4. Use UI field mapping to wire frontend/API requests to CRs.
